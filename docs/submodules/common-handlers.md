| Path | Repository | Targets |
|---|---|---|
| handlers/common/file | xmip-handler-file | server, edge, desktop |
| handlers/common/ftp | xmip-handler-ftp | server, edge, desktop |
| handlers/common/http | xmip-handler-http | server, edge, desktop, iot |
| handlers/common/grpc | xmip-handler-grpc | server, edge, desktop |
| handlers/common/web-api | xmip-handler-web-api | server, edge, desktop |
| handlers/common/soap | xmip-handler-soap | server, edge |
| handlers/common/tcp | xmip-handler-tcp | server, edge, desktop, iot |
| handlers/common/udp | xmip-handler-udp-base | server, edge, desktop, iot |
