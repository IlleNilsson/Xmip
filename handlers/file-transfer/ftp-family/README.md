# FTP Family Handler

Placeholder for the shared FTP family handler repository.

This repository covers:

- FTP
- FTPS
- SFTP

These technologies share enough concepts to live together:

- connection configuration
- credentials and identity
- directory listing
- file pickup
- file send
- file naming
- retries
- scheduling or polling
- stream handoff to Xmip

Protocol-specific behavior shall be implemented behind shared FTP family abstractions.
