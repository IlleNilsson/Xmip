# Web API Handler

Placeholder for the Web API handler repository.

Web API is derived from the base HTTP handler.

This repository owns Web API specific behavior:

- API endpoint definitions
- request binding
- response binding
- route to Xmip message mapping
- OpenAPI or similar metadata where applicable
- API-specific authentication and authorization policy
- API versioning
- API error mapping

Shared HTTP primitives and shared HTTP security belong in the base HTTP handler repository.
