# SOAP Handler

Placeholder for the SOAP handler repository.

SOAP is derived from the base HTTP handler when transported over HTTP.

This repository owns SOAP specific behavior:

- SOAP envelope handling
- SOAP headers
- SOAP body mapping
- WSDL support where applicable
- SOAP action routing
- fault mapping
- XML contract integration
- SOAP-specific authentication and authorization policy

Shared HTTP primitives and shared HTTP security belong in the base HTTP handler repository.
